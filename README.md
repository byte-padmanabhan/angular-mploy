# MPloyChek Role-Based Verification Dashboard

## Overview

This project is a full-stack role-based verification dashboard built as part of the NSQTech / MPloyChek internship challenge.

The application demonstrates:

- JWT Authentication
- Role-Based Access Control
- Angular Standalone Components
- Express.js REST APIs
- Async API Processing
- Modular Frontend Architecture
- Admin User Management
- Protected Backend Routes

---

## Tech Stack

### Frontend
- Angular 20+
- TypeScript
- Angular Router
- Angular Forms
- Standalone Components
- HttpClient

### Backend
- Node.js
- Express.js
- JWT Authentication
- Middleware-based Authorization
- Mock JSON/In-Memory Database

---

## Features

### Authentication
- Login using:
  - User ID
  - Password
  - Role
- JWT token generation
- Protected routes
- Role validation

### User Dashboard
General users can:
- View their profile
- View their own records
- Experience simulated async API loading

### Admin Dashboard
Admins can:
- View all records
- View all users
- Create users
- Delete users
- Manage role-based access
